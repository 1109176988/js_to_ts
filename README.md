# js_to_ts 目前代码只有非常简单的js可以进行转化, 欢迎朋友们查看
使用nodejs将简单的js代码转化成为ts代码
案例: (js转化为ts的步骤, 下载代码, 进入node.js代码所在文件, 在terminal执行node node.js, 执行成功后即可在terminal看到输出: 执行成功了)
